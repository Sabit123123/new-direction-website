Сюда кладите 3D-модели оборудования в формате GLB (или GLTF/OBJ/FBX/STL).

ВАЖНО: файлы .skp (SketchUp) сюда НЕ подходят — браузер их не открывает.
Сначала экспортируйте каждую модель из SketchUp в GLB:
  SketchUp → File → Export → 3D Model → формат glTF (.glb).

Называйте файлы латиницей без пробелов, например: cdj3000.glb, m32live.glb, mac_aura.glb.
После этого я пропишу каждую модель к нужной позиции каталога (catalog.js, поле model),
либо вы сами загрузите её в планировщике кнопкой «Загрузить 3D-модель» у выбранного объекта.
